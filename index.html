VedicWell

"AI-powered Ayurvedic wellness platform solving fragmented digital health challenges through dosha-based personalization, local nutrition planning, and intergenerational wisdom exchange."

Problem Statement

"Fragmented wellness apps fail to deliver personalized Ayurvedic guidance based on individual dosha constitution, forcing users to juggle multiple generic solutions."

Literature Review / Market Research

  • Headspace: Meditation only, 25M users, no Ayurveda
  • MyFitnessPal: Generic calorie tracking, 200M users, ignores doshas
  • Flo: Women's health tracker, lacks cultural context
  • AyuRythm: Basic Ayurvedic app, recipe-focused, no AI
  • Jiva Ayurveda: Teleconsultation only, no personalization engine
  • NirogStreet: Symptom checker, lacks dosha analysis
  • GAP: No unified AI+Ayurveda platform with dosha assessment (92% accuracy target), local nutrition planning, and intergenerational wisdom exchange

Research Gap / Innovation

  • No unified AI platform combines dosha assessment (92% accuracy) with local ingredient nutrition planning and intergenerational practitioner wisdom exchange
  • Missing real-time Ayurvedic symptom analysis using NLP trained on Charaka Samhita mappings
  • Fragmented solutions force users across 6+ apps instead of holistic wellness ecosystem
  • Lack of evidence-based ML models validated against practitioner assessments
  • No mobile-first PWA with offline dosha tracking and cultural UI design system

System Methodology

Dataset / Input

  • Doshas for Vata, Pitta, and Kapha (5K+ records)
  • AyurGenixAI Ayurvedic Dataset (50K+ symptoms)

55K+ total records with preprocessing for feature extraction and model training

Model / Architecture

XGBoost Dosha Classifier (92% accuracy) + GPT-4 NLP Symptom Analysis + Hybrid Recommendation Engine

Workflow:

  • 50+ assessment questions → Feature extraction (NLP + Numerical)
  • XGBoost ensemble → Vata/Pitta/Kapha % prediction
  • GPT-4 → Symptom → Charaka Samhita remedy mapping
  • Collaborative filtering → Personalized nutrition/rituals
  • LangChain → Real-time workflow orchestration

Tech Stack:

  • Frontend: React/Next.js + TailwindCSS (PWA)
  • Backend: Node.js FastAPI + PostgreSQL/Redis
  • AI: Pinecone VectorDB + AWS SageMaker endpoints
  • Deployment: Vercel + AWS Lambda (99.9% uptime)
